# alfredo.design Archive

Static archive backup for [alfredo.design](https://alfredo.design).

## Purpose

This repository contains a **static HTML archive** of the alfredo.design blog content. It serves as a backup and historical reference.

## Contents

- `index.html` - Static HTML snapshot of the site
- `feed.json` - JSON feed of blog posts
- `uploads/` - Media files and assets
